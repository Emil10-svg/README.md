### Equity Research Portfolio

This repository contains my personal equity research work focused on fundamental analysis, valuation,
and small-cap screening tools.

### About Me
I am a high school student (Germany) with a strong interest in buy-side investing, company valuation,
and building automated screening tools using Python and Excel.

### What You Will Find Here

### Investment Memos
Located in `/memos/`.
Each memo is a 2-page research note including:
- business model overview
- key financial metrics
- valuation (multiples / EPV / DCF where applicable)
- risks and catalysts

### Excel Valuation Models
Located in `/excel_models/`.
Includes:
- DCF valuation template
- EPV (Earnings Power Value) model
- company snapshot templates (income statement, balance sheet, cashflow)

### Python Screening Tools
Located in `/python_tools/`.
Includes a fundamental screener that:
- imports a list of tickers
- pulls financial data
- calculates valuation and quality metrics
- ranks companies using a scoring system
- exports results to Excel/CSV

### Datasets
Located in `/datasets/`.
Contains sample ticker lists and example screener outputs.

### Key Metrics Included
- EV/EBITDA
- P/E
- Free Cashflow
- Revenue and FCF trends
- Debt ratios
- Altman Z-Score (where available)
- EPV valuation framework
